## About The Project

### Built With

- rust
- gtk4
- libadwaita
- nix flake

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   cd hotkeys
   ```
3. start a nix flake shell
   ```sh
   nix develop
   ```
4. run Cargo
   ```sh
   cargo run
   ```
